Бот для моего сервера в Discord. Собирает все сообщения в чате и сохраняет, после чего с шансом 10% цитирует одно сообщение из сохранённых. Иногда так фантастически угадывает, что аж смешно.

<h2> Общая информация </h2>

* Этот репозиторий - проект Gradle, который должен быть открыт через IntelliJ IDEA или импортирован в Eclipse IDE с плагином Kotlin.
* Используемая версия Gradle - 8.3.
* Используемая версия Kotlin - 1.8.22
* Используемая версия JDK - Eclipse Temurin 1.8.0_382.

<h2> Использование </h2>

Запустить скомпилированный jar-файл. Там присутствует GUI.
